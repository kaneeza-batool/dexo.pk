# DEXO.PK

Multi-page e-commerce frontend for a luxury home decor brand. Built with a focus on clean layout, responsive design, and smooth user experience across all devices.

**Live Demo:** [dexopk.netlify.app](https://dexopk.netlify.app)

## Features

- Product listing with category sections
- Responsive across mobile, tablet, and desktop
- Clean luxury aesthetic with consistent styling
